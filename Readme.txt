Project Title: Course Recommender

Prerequisites: 

Needs installation of :
1. R (R-3.3.2 for Windows (32/64 bit)) 
Link :https://cran.r-project.org/bin/windows/base/

2. RStudio 1.0.44 - Windows Vista/7/8/10		
   RStudio 1.0.44 - Mac OS X 10.6+ (64-bit)
Link:https://www.rstudio.com/products/rstudio/download3/

Packages required :
1. arules
2. shiny
3. shinydashboard
4. stringr
5. datasets


Steps to run the program:

1. Place the ui.r, server.r, Login.r file and all csv files (databases) in the same directory

2.Set the working directory as above using the command setwd('path/to/directory')

3.Open server.r/ui,r and click runApp button on the top right of the editor
